2022-04-22
  1. Java程序的开始点是由主方法开始的。
  2. JAVA语言的注释方式有以下3种。
    • 单行注释：//
    • 多行注释： /*… */
    • 文档注释：/* *…… */
  3. Java中变量名称可以由字母、数字、下划线、和美元符组成，但是标识符不能以数字开头，也不能是Java中的关键字。此外，Java变量有大小写之分。
  4. Java基本组成是类，使用public class和class都可以申明一个类，但是前者类名必须与文件名一致，后者文件名可以与类名不一致，但是执行时必须执行生成后的* .class文件。
  5. goto和const是未使用的两个关键字。
  6. assert和enum 是jdk新版本中添加的关键字。
  7. System.out.print()实在标准输入设备——————显示器上进行输出操作，后面可以使用println和print两种输出方法，前者是在输出之后加入换行，后者没有换行。
